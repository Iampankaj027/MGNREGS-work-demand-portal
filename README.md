# NREGA Job Portal

## Overview

The NREGA Job Portal is a comprehensive web-based application designed to facilitate rural employment opportunities under the National Rural Employment Guarantee Act (NREGA). This portal allows users to apply for various job projects, track application status, and provides administrative tools for managing job postings and applications. Built as a single-page application using modern web technologies, it offers a seamless experience for job seekers and administrators alike.

## Features

### For Job Seekers
- **Easy Job Application**: Submit applications for rural development projects with a user-friendly form
- **Location-Based Selection**: Choose from multiple states, districts, and villages across India
- **Application Tracking**: Monitor application status and view detailed information
- **Real-Time Statistics**: View dashboard with total applications, pending, approved, and active jobs
- **Responsive Design**: Optimized for desktop and mobile devices

### For Administrators
- **Admin Dashboard**: Comprehensive panel for managing applications and system operations
- **Bulk Operations**: Approve multiple applications at once or perform quick approvals
- **Data Management**: Add sample data, clear all data, or export application records
- **Reporting**: Generate detailed reports on application statistics and breakdowns
- **Application Oversight**: View recent applications and update statuses

### Technical Features
- **Offline Capability**: Uses localStorage for data persistence (no backend required)
- **Progressive Web App (PWA)**: Includes PWA meta tags for potential mobile app experience
- **Modern UI**: Styled with Tailwind CSS for a clean, professional interface
- **Interactive Elements**: Dynamic location selection, search, filtering, and pagination
- **Confirmation Modals**: User feedback for successful submissions and actions

## Technologies Used

- **HTML5**: Structure and semantic markup
- **CSS3**: Styling with Tailwind CSS framework
- **JavaScript (ES6+)**: Client-side logic and interactivity
- **Local Storage API**: Data persistence
- **Tailwind CSS**: Utility-first CSS framework for responsive design
- **Inter Font**: Modern typography from Google Fonts

## Installation and Setup

1. **Clone or Download**: Obtain the project files
2. **Open in Browser**: Simply open index.html in any modern web browser
3. **No Dependencies**: The application runs entirely in the browser with no server setup required

## Usage

### Getting Started
- Open index.html in your web browser
- Navigate through the portal using the header menu

### Applying for Jobs
1. Click "Apply for Job" from the landing page
2. Fill out the application form with personal details, location, and work preferences
3. Submit the form to receive an application ID

### Checking Status
1. Click "Check Status" from the landing page
2. Use search by Application ID or filter by status
3. View paginated results with detailed application information

### Admin Access
1. Click "Admin" from the header
2. Use various admin tools to manage applications and generate reports

## Data Structure

The application includes predefined location data for major Indian states including:
- Andhra Pradesh, Bihar, Gujarat, Karnataka, Maharashtra, Rajasthan, Tamil Nadu, Uttar Pradesh, West Bengal
